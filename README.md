## 🚀 Features

- **Modern UI:** Sleek, professional layout showcasing skills, projects, and achievements.
- **Lightning Fast:** Optimized performance for an exceptional browsing experience.
- **Responsive Design:** Mobile-friendly for seamless viewing on any device.
- **Clean Code:** Written in TypeScript for maintainability and scalability.

## 🌟 Get Started

1. Clone the repo:
   ```bash
   git clone https://github.com/PIYUSH0-7/PIYUSH07-PORTFOLIO-.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm start
   ```
